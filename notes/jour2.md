# Jour 2 – Linux essentiel

## Objectifs
- Comprendre le terminal Linux
- Créer des dossiers et fichiers
- Lire et rechercher du texte
- Comprendre les permissions

## Commandes apprises
pwd, ls, cd, mkdir, touch, cat, less, grep, chmod

## Exercices faits
- Création du dossier linux-lab
- Création de fichiers texte ex1.txt à ex10.txt
- Lecture de fichiers avec cat et less
- Test des permissions avec test.sh
- Recherches de mots avec grep

## Difficultés
- Comprendre les permissions au début
- Savoir où exécuter les scripts

## Ce que j’ai compris
- Linux fonctionne avec des commandes simples
- grep permet de chercher un mot dans un fichier
- chmod +x permet d’exécuter un script
- nano sert à créer ou modifier un fichier dans le terminal
# Fiche personnelle – 20 commandes Linux utiles
1. pwd       # Affiche le chemin actuel
2. ls        # Liste les fichiers dans le dossier
3. cd        # Changer de dossier
4. mkdir dir # Crée un dossier nommé dir
5. touch f   # Crée un fichier nommé f
6. cp f1 f2  # Copie f1 vers f2
7. mv f1 f2  # Déplace/renomme f1 vers f2
8. rm f      # Supprime le fichier f
9. cat f     # Affiche le contenu de f
10. less f   # Affiche le contenu de f page par page
11. head f   # Affiche les 10 premières lignes de f
12. tail f   # Affiche les 10 dernières lignes de f
13. tail -f f # Suit un fichier en temps réel
14. grep mot f  # Cherche le mot dans le fichier f
15. find . -name "*.txt" # Cherche tous les fichiers .txt
16. chmod +x f # Rend le fichier f exécutable
17. chown user f # Change le propriétaire du fichier f
18. whoami   # Affiche ton nom d’utilisateur
19. date     # Affiche la date et heure
20. echo "texte" # Affiche le texte dans le terminal
