| POST    | Créer une ressource | /posts | 201 Created |
| PUT     | Modifier une ressource | /posts/1 | 200 OK |
| DELETE  | Supprimer une ressource | /posts/1 | 200 OK ou 204 No Content |
| POST     |Créer un post  |/posts  |201 Created  |
| PUT |Modifier un post |/posts/1 |200 OK |
| DELETE |Supprimer un post|/posts/1 |200 OK ou 204 No Content |
| GET     | Lire tous les posts | /posts        | 200 OK  |
